The NYCCSC Map Viewer

This is a test / staging release of the Map viewer"

The files below are the deliverables.

  • Harness -- The test harness which lets you test a map-viewer iframe. This will default to the just released map viewer. It can also run the stub by setting the URL appropriately
  • Meta Data -- The meta-data for what the map is displaying.
  • Stub Map Viewer -- The test stub. This implements the API for the map viewer and reports in it's window the current parameters that have been set.
  • Map Viewer -- The map viewer. Using this link opens the map viewer with all default settings.
